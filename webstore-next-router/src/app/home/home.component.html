Welcome to IPT WebStore Demo! SPA using Angular 7, RxJS 6, Material Design, and Typescript

IPT Webstore Demo shows how to build a single page appplication using Angular 2 and Angular Router, HTTP Client, Forms, etc. modules. It is written completely in TypeScript and utilizes reactive event processsing using RxJS client side library. Styling is done using Angular Material and Material Flexbox. Application demonstrates following functionality:

  • Anonymous users can browse products and view detailed information about each product.
  • Registered users can browse products and add them to the shoping basket
  • Operators can edit product information and add new products to the catalog.
  • Administrators can manage (create, read, update and delete ) all registered users, and products.
