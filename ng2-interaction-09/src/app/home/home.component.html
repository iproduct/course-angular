IPT Product Catalog Demo

SPA using Angular 2, RxJS, Bootstrap, and Typescript

Learn More @ Github Wiki

IPT Product Catalog demonstartes how to build a single page appplication using Angular 2 and Angular Router, HTTP, Forms, etc. modules. It is written completely in TypeScript and utilizes reactive event processsing using RxJS client side library. Styling is done using Bootstrap.

Application provides following functionality:

  • Anonymous users can browse products and view detailed information about each product.
  • Registered users can browse products and add them to the shoping basket
  • Operators can edit product information and add new products to the catalog.
  • Administrators can manage (create, edit user data and delete ) all registered users, as well as products.
